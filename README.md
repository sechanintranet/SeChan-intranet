# 세찬 해피콜 관리시스템 V13 D+95/D+185 Manager Assign

## 반영
- D+95 / D+185 전용 배정 로직 추가
  1. 판매자가 현재 재직중이면 판매자 본인 배정
  2. 판매자가 퇴사자면 근무이력에서 개통일 당시 해당 매장의 점장 검색
  3. 당시 점장이 현재 재직중이면 당시 점장 배정
  4. 당시 점장이 퇴사자이거나 없으면 현재 해당 매장의 재직중 점장 배정
  5. 그래도 없으면 배정불가
- 월간정기 / D+1 / D+7 / D+13은 기존 배정 로직 유지
- employee_store_history의 role=점장, start_date/end_date 기준 사용
