2. 직원별 고객 악세사리 주문 생성

   - 매장 도착
7. 주문 완료 체크 시 도착 예정 날짜 필수

   - 주문 완료건
   - 매장 도착건
   - 본인 총 주문 금액
10. 점장용 매장 현황
   - 매장 직원별 주문 확인
   - 카테고리/인원별 필터
   - 매장 총 주문 금액
11. 관리자 이상 전체 현황

   - 매장별
