1. 주문건 1개에 여러 품목 입력 가능
2. 품목별 카테고리/품목명/금액 입력
3. 주문 원가 합계 자동 계산
4. 고객에게 받기로 한 금액 추가
   - 공란이면 무료제공으로 자동 분류
5. 상태 추가
   - 주문 미완료
   - 주문 완료
   - 매장 도착
   - 고객 수령
   - 반품
6. 반품 사유 추가
7. 직원/점장도 내 주문과 매장 현황 모두 조회 가능
8. 관리자 이상은 전체 현황 조회 가능
9. 매장 현황에서 담당자/카테고리/주문경로 필터 가능
10. 내 주문건 삭제 기능 추가
11. 악세사리 주문 UI를 해피콜 방식 탭 구조로 개선
