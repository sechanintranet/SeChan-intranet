# 세찬 해피콜 관리시스템 V11.2 Clean Employee UI

## 수정
- 직원관리 메뉴를 상단 독립 메뉴로 재배치
- 직원관리 메인 표 단순화
- 입사일/퇴사일/근무이력은 상세 팝업 안으로 이동
- 근무이력 버튼을 상세 버튼으로 통합
- 메인 표 컬럼을 이름/매장/상태/권한/비밀번호/상세/저장 중심으로 정리
- 메뉴 아코디언을 더 작고 덜 가리는 컴팩트 구조로 변경

## 유지
- 재직중/퇴사자 탭

- 근무이력 저장/삭제
